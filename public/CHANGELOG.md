# Nexora Release History 🚀

All the latest updates for your favorite habit-building bro!

## [1.5.2] - 2026-04-17
### Added
- **Smart Build System**: Support for "Public" version files to trigger notifications.
- **Furniture Layering**: Moved items now automatically jump to the top of the stack for better design control.

### Fixed
- **Stable ID Tracking**: Critical fix for the "Messy Furniture" bug. Furniture now stays perfectly preserved even when switching between isometric/miniature/cozy house styles.
- **Persistence Bridge**: Improved Firestore sync logic to prevent accidental data overwrites on slow connections.

## [1.5.1] - 2026-04-16
### Added
- **Nuclear Refresh**: Implemented a robust system to clear PWA cache and service workers during updates.
- **Water Challenge v2**: Added repetitive sound effects and mascot reactions to hydration goals.

### Fixed
- **Mascot Pinning**: Fixed a bug where the mascot would "float" away from pinned furniture after a reload.
- **Firebase Auth Persistence**: Ensured users stay logged in across app restarts.
